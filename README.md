项目简介
本项目旨在构建一个面向学术文献的智能检索增强代理，通过大语言模型对文献内容进行深度理解，自动生成结构化检索画像与增强式检索语句，提升文献检索的准确性与相关性。

系统核心功能：
- 从论文标题、摘要、引言、结论中自动抽取研究主题
- 通过通义千问大模型生成结构化检索描述与关键词
- 构建原始查询与 LLM 增强查询的对比与保存机制
- 提供轻量级 Web 接口，便于后续扩展为完整检索系统

 项目结构

 环境配置
1. 安装依赖
pip install -r requirements.txt

 2. 配置 API Key
复制 `.env.example` 为 `.env`，并填入通义千问 API Key：
DASHSCOPE_API_KEY=your_api_key_here

4. 运行项目
python app.py

核心流程
1. 输入一篇学术文献（标题、摘要、全文等）
2. LLM 智能分析文献内容，生成检索描述 + 关键词
3. 自动构建增强检索查询
4. 保存原始查询与增强查询对比结果
5. 输出可直接用于学术检索引擎的高质量检索式
